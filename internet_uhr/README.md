# Internet Uhr

## Project Description

Mache die anzeige der Uhrzeit (Landscape) und benutze dazu möglichst den ganzen Bildschirm und achte auf eine schöne Darstellung mit "Schönen" Zahlen. 
Die datei Credentials.h -> #include <Credentials.h> stellt die Wifi zugangsdaten im Format const char* ssid = "xxxx";
const char* password = "yyyyy";
bereit

Verwende die LovyanGFX mit #include <CYD_Display_Config.h> um eine LGFX Klasse zu erzeugen (lgfx::LGFX_Device)
